Q: What is a video game?
A: A video game, or computer game, is a software program in which on or more players make decisions through the control of game objects and resources, in pursuit of a goal.

Q: What are game engines?
A: A game engine is a software framework primarily designed for the development of video games, and generally includes relevant libraries and support programs. The "engine" terminology is similar to the term "software engine" used in the software industry.

Game engine can also refer to the development software utilizing this framework, typically offering a suite of tools and features for developing games.

Developers can use game engines to construct games for video game consoles and other types of computers. The core functionality typically provided by a game engine may include a rendering engine ("renderer") for 2D or 3D graphics, a physics engine or collision detection (and collision response), sound, scripting, animation, artificial intelligence, networking, streaming, memory management, threading, localization support, scene graph, and video support for cinematics. Game engine implementers often economize on the process of game development by reusing/adapting, in large part, the same game engine to produce different games or to aid in porting games to multiple platforms.

Web Development

Q: What is the web?
A: The web is a distributed service which provides multimedia information based on hypertext documents. Distributed as in the source of the information is spread across multiple servers around the world. Multimedia as in the information comes in various forms, such as text, images, audio, video. Hypertext is text displayed on a computer display or other electronic devices with references to other text that the reader can immediately access. Hypertext documents are interconnected by hyperlinks, which are typically activated by a mouse click, keypress set, or screen touch.

Q: Tell me about IP addresses.
A: IP stands for Internet Protocol, a set of rules that govern how communicating across the Internet works. Loosely speaking, an IP address identifies a destination on the Internet to send data to and from. IPv4 has 32bits of data, segmented into four 8-bit numbers.

Q: What are domain name systems for?
A: A domain name is an easy to write text that identifies a company's or a service's website on the Internet, as opposed to using an IP address. DNS servers store a mapping between the domain name and the actual IP address that it maps to.

Q: What is HTML?
A: Hypertext Markup Language is used to build Web pages, which contains hyperlinks which can point to other parts of hypertext documents. It also contains information for Web browsers to render a Web page. A Web page consists of HTML tags which structure the data throughout a Web page, and the actual content itself to be structured.
